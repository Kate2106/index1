Задание 7

Написать функцию, для проверки правильности заполнения поля "номер телефона" в форме. Правильным считается, если поле:   заполнено (не пусто); содержит только цифры и знаки "-"; знаки "-" не идут подряд и не стоят в начале.конце номера; Далее Вам необходимо создать форму с именем "Test", в которой помещены:   поле ввода с именем "In"; поле ввода с именем "Result"; кнопка "Enter", которая при нажатии должна вызывать Вашу функцию, передав ей в качестве аргумента строку из поля "In", а результат работы функции вывести в поле "Result". Более конкретно, обработка нажатия кнопки должна выглядеть так: onclick="document.getElementById('Result').value=имя_функции(document.getElementById('In') .value)" где имя_функции необходимо заменить на имя Вашей функции. В поле "Result" следует выводить слово "Okay", если проверка прошла успешно и "Fail" в противном случае.

Ввести номер телефона
